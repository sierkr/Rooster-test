Rooster v3.27.111 — Release notes
==================================

GEEN BACKUPS IN DE TESTOMGEVING
-----------------------------
In de testomgeving kan geen backup meer gemaakt worden. Klik je daar op "Nu
backup maken", dan verschijnt de melding dat dit in de testomgeving niet kan.
Zo kan er nooit een testbackup ontstaan die je per ongeluk in de live-agenda
terugzet.

- De automatische backup vóór een Excel-import wordt in test overgeslagen.
- Een backup terugzetten kan in test wél: maak een backup in de live-agenda en
  herstel die in test om met actuele data te oefenen.
